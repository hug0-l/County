## BE-1: Backend Core + NTP Service — Completed

### Created Files
- **`server.py`** — FastAPI single-file server with:
  - CORS middleware (all origins)
  - SQLite DB at `vcc_pre.db` (auto-created, 4 tables: schedules, presets, config, ntp_logs)
  - `NTPManager` class using ntplib connecting to `time.hko.hk` (UDP 123)

  - Auto-opens browser at `http://localhost:8000` via `threading.Timer`
  - Serves existing SPA via `FileResponse` (avoids Jinja2 compatibility issue with Starlette)
- **`requirements.txt`** — fastapi, uvicorn, ntplib, jinja2, python-multipart
- **`templates/index.html`** — copy of `VPre_CountdownCtrl_PROD_v0.5.html`

### Dev Note
Used `FileResponse` instead of Jinja2 templates because Starlette 1.3.x + Jinja2 has a `TypeError: unhashable type: 'dict'` bug with the template `globals` argument. Since the SPA is fully static HTML, `FileResponse` is the clean fix.
